# 普通用户使用说明

本文适用于当前默认角色中的“普通用户”。实际可见菜单和按钮受角色授权影响，如果管理员调整了菜单或 API 权限，以系统当前展示为准。

## 1. 登录与个人中心

1. 打开系统登录页，输入管理员分配的账号和密码。
2. 登录成功后默认进入“工作台”。
3. 右上角头像菜单可进入“个人资料”或退出登录。
4. “个人资料”中可查看个人信息，并可在“修改密码”页签中修改登录密码。

说明：当前默认普通用户主要用于学习、答题和练习。如果在修改个人资料、切换开关或保存配置时提示“没有权限”或 403，表示当前角色没有对应写入权限，请联系管理员处理。

## 2. 工作台

普通用户工作台展示个人学习相关信息：

- 可参加考试：当前已发布且启用的试卷数量。
- 答题中：尚未提交的答卷数量。
- 等待阅卷：已提交但需要人工阅卷的答卷数量。
- 练习正确率：历史练习中的整体正确率。

工作台的快捷入口包括：

- 我的答题
- 练习中心
- 知识文章

可点击列表中的“可参加考试”或“最近答题记录”进入对应页面继续操作。

## 3. 我的答题

路径：`答题管理 / 我的答题`

### 3.1 查看可答试卷

1. 进入“我的答题”。
2. 在“可答试卷”页签中查看可参加的考试。
3. 可通过试卷标题搜索。
4. 点击“开始答题”进入答题页；如果已有未提交答卷，按钮会显示“继续答题”。

### 3.2 作答与暂存

答题页支持以下题型：

- 单选题
- 多选题
- 判断题
- 填空题
- 简答题

作答过程中可以点击“暂存答卷”保存当前答案，稍后再继续。

### 3.3 提交答卷

1. 确认答案无误后点击“提交答卷”。
2. 提交后不能再修改答案，也不能重复参加同一份试卷。
3. 客观题会自动判分。
4. 如果试卷包含填空题或简答题，答卷会进入“待阅卷”状态，等待管理员人工评分。
5. 阅卷完成后，可在“我的记录”中查看结果、参考答案、解析和得分。

## 4. 练习中心

路径：`答题管理 / 练习中心`

### 4.1 生成练习

1. 点击“添加题目”。
2. 可按题库、题型、难度、题干关键字筛选题目。
3. 勾选题目，或填写数量后点击“随机勾选”。
4. 点击“加入练习”。
5. 回到练习中心后点击“开始练习”。

### 4.2 练习作答

1. 左侧显示题目列表、完成进度、正确数和错误数。
2. 完成当前题后点击“提交并看答案”。
3. 系统会展示我的答案、正确答案和解析。
4. 点击“下一题”继续。
5. 可点击“完成练习”提前结束。

### 4.3 练习记录

练习中心下方展示历史练习记录，可进行：

- 查看或继续未完成练习
- 重新练习同一组题目
- 删除练习记录

## 5. 知识文章

路径：`知识库管理 / 知识浏览`

1. 左侧为知识目录，按分类展示已发布文章。
2. 可按文章标题搜索。
3. 点击文章后在右侧查看正文、摘要、标签、发布时间和内容文件。
4. 只有已发布且分类启用的文章会出现在知识浏览中。

当前默认普通用户以浏览知识为主。若看到“知识分类”或“知识文章”等管理页面，通常只能查看列表，不能新建、编辑、删除、发布或置顶。

## 6. 只读管理入口

当前默认初始化权限下，普通用户可能会看到部分系统管理菜单，并具备 GET 查询权限，例如用户、角色、菜单、API、部门、审计日志等列表。

普通用户不应执行系统配置类操作，包括但不限于：

- 新建、编辑、删除用户
- 重置他人密码
- 新建、编辑、删除角色
- 保存角色菜单和接口权限
- 新建、编辑、删除菜单或 API
- 新建、编辑、删除部门

如果页面上出现开关或操作入口，但点击后提示无权限，这是当前角色的权限边界，不影响正常答题、练习和知识浏览。

## 7. 常见状态说明

- 答题中：答卷已创建但尚未提交，可继续作答。
- 待阅卷：已提交，包含主观题，等待管理员人工评分。
- 已阅卷：评分完成，可查看得分和结果。
- 草稿试卷：未发布，普通用户不可作答。
- 已发布试卷：可在“我的答题”中参加。
- 已关闭试卷：不再开放作答。
