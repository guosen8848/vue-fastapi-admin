# 管理员使用说明

本文适用于当前默认角色中的“管理员”，以及开启了“超级用户”的账号。当前系统中，超级用户会绕过接口权限校验；“管理员”角色默认拥有全部菜单和 API 权限。实际可见功能仍以系统当前角色授权为准。

## 1. 登录与工作台

1. 使用管理员账号登录系统。
2. 登录后默认进入“管理员工作台”。
3. 工作台展示题库数量、题目数量、已发布试卷、待阅卷数量。
4. 工作台提供题库管理、题目管理、试卷管理、阅卷中心快捷入口。
5. 右上角头像菜单可进入个人资料、修改密码或退出登录。

## 2. 系统管理

### 2.1 用户管理

路径：`系统管理 / 用户管理`

可完成以下操作：

- 按用户名、邮箱搜索用户。
- 按部门树筛选用户。
- 新建用户，填写用户名、邮箱、密码、角色、部门等信息。
- 编辑用户资料、角色、部门、启用状态、超级用户状态。
- 禁用或启用用户。当前登录用户不能禁用自己。
- 删除用户。
- 重置非超级用户密码，默认重置为 `123456`。

### 2.2 角色管理

路径：`系统管理 / 角色管理`

可完成以下操作：

- 新建、编辑、删除角色。
- 点击“设置权限”配置角色的菜单权限和接口权限。
- 在“菜单权限”中勾选该角色可见的页面。
- 在“接口权限”中勾选该角色可调用的 API。
- 点击“确定”保存授权。

权限调整后，建议让相关用户重新登录或刷新页面，以重新加载菜单和按钮权限。

### 2.3 菜单管理

路径：`系统管理 / 菜单管理`

可维护前端动态菜单：

- 新建根菜单或子菜单。
- 配置菜单名称、访问路径、组件路径、跳转路径、图标、排序。
- 设置菜单隐藏或 KeepAlive。
- 编辑或删除菜单。

注意：存在子菜单的菜单不能直接删除，需要先删除子菜单。

### 2.4 API 管理

路径：`系统管理 / API管理`

可完成以下操作：

- 查看和搜索 API。
- 新建、编辑、删除 API。
- 点击“刷新API”，根据后端已注册路由同步 API 列表。

新增后端接口后，应刷新 API，并在角色管理中为对应角色分配接口权限。

### 2.5 部门管理

路径：`系统管理 / 部门管理`

可完成以下操作：

- 新建部门。
- 编辑部门名称、备注、排序和父级部门。
- 删除部门。

### 2.6 审计日志

路径：`系统管理 / 审计日志`

可按用户名称、功能模块、接口概要、请求方法、请求路径、状态码、操作时间筛选日志，并查看请求体和响应体。

## 3. 知识库管理

### 3.1 知识分类

路径：`知识库管理 / 知识分类`

可完成以下操作：

- 新建分类，支持根分类和二级分类。
- 编辑分类名称、说明、排序、启用状态。
- 删除分类。

注意：

- 知识分类仅支持两级结构。
- 有子分类的分类不能删除。
- 分类下存在文章时不能删除。

### 3.2 知识文章

路径：`知识库管理 / 知识文章`

可完成以下操作：

- 按标题、分类、状态筛选文章。
- 新建文章，填写标题、分类、摘要、标签、状态。
- 上传内容文件，支持 `.md`、`.docx`、`.pdf`、`.xlsx`。
- 调整内容文件顺序，或删除内容文件。
- 编辑文章。
- 在“草稿”和“已发布”之间切换。
- 设置或取消置顶。
- 删除文章。

已发布且所在分类启用的文章，会出现在 `知识库管理 / 知识浏览` 中。

## 4. 答题管理

### 4.1 题库管理

路径：`答题管理 / 题库管理`

可完成以下操作：

- 查看题库列表，按题库名称和启用状态筛选。
- 下载题库模板。
- 导入 `.xlsx` 题库文件。
- 新建、编辑、删除题库。
- 启用或停用题库。

导入模板固定使用 `questions` Sheet，表头包括题型、题目分类、题干、选项、正确答案、分值、难度、解析、参考答案、标签等字段。导入失败时，系统会返回行号、字段和错误原因。

注意：题库中的题目如果已被试卷引用，该题库不能直接删除。

### 4.2 题目管理

路径：`答题管理 / 题目管理`

可维护以下题型：

- 单选题
- 多选题
- 判断题
- 填空题
- 简答题

可完成以下操作：

- 按题库、题型、难度、启用状态、题干关键字筛选题目。
- 新建题目，配置题干、选项、正确答案、参考答案、解析、标签、难度、分值。
- 编辑题目。
- 启用或停用题目。
- 删除题目。

注意：已被试卷引用的题目不能删除。

### 4.3 试卷管理

路径：`答题管理 / 试卷管理`

可完成以下操作：

- 按试卷标题、试卷状态、启用状态筛选试卷。
- 新建试卷，配置标题、说明、状态、限时分钟、及格分、启用状态。
- 添加题目，可按题库、题型、难度、题干关键字筛选，也可随机勾选。
- 调整题目顺序、移除题目、设置每题分值、打乱题序。
- 编辑试卷。
- 发布试卷或关闭试卷。
- 启用或停用试卷。
- 查看“作答情况”，包括作答人数、已提交、待阅卷、已阅卷、平均分、最高分、及格率和答题人明细。
- 删除试卷。

注意：

- 只有已发布且启用的试卷会出现在普通用户“我的答题”中。
- 已有答卷的试卷不能调整题目集合。
- 已有答卷记录的试卷不能删除。

### 4.4 阅卷中心

路径：`答题管理 / 阅卷中心`

用于处理包含填空题或简答题的待阅卷答卷。

操作流程：

1. 按试卷标题、答题人、状态筛选阅卷任务。
2. 点击“领取”锁定当前答卷。
3. 进入“阅卷”详情。
4. 查看题干、用户答案、参考答案和解析。
5. 对主观题填写人工得分和阅卷备注。
6. 点击“保存评分”。
7. 所有主观题评分完成后，点击“完成阅卷”。

说明：

- 客观题由系统自动判分，阅卷中心只展示自动得分。
- 待阅卷答卷需要先领取后评分。
- 已被其他管理员领取的答卷不能直接评分。
- 可释放自己领取的答卷，让其他管理员继续处理。

### 4.5 我的答题与练习中心

管理员默认也拥有“我的答题”和“练习中心”的权限，可用于自测流程：

- 在“我的答题”中参加已发布试卷。
- 在“练习中心”中选择题目生成练习，提交后查看答案和解析。

## 5. 推荐使用流程

1. 在“部门管理”维护组织结构。
2. 在“角色管理”维护角色及菜单、接口权限。
3. 在“用户管理”创建用户并分配角色。
4. 在“知识分类”和“知识文章”维护知识内容。
5. 在“题库管理”导入或创建题库。
6. 在“题目管理”检查和维护题目。
7. 在“试卷管理”组卷并发布。
8. 普通用户完成答题。
9. 在“阅卷中心”处理待阅卷答卷。
10. 在“试卷管理 / 作答情况”和“审计日志”查看结果与操作记录。

## 6. 常见状态说明

- 草稿：试卷或文章未发布，普通用户不可作答或浏览。
- 已发布：试卷可作答，文章可浏览。
- 已关闭：试卷不再开放作答。
- 答题中：用户已开始但未提交。
- 待阅卷：答卷已提交，存在需要人工评分的题目。
- 已阅卷：答卷评分完成。
- 启用：资源可用于前台选择或展示。
- 停用：资源不再用于普通用户作答或浏览。
